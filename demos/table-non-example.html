Operating Systems Accessibility Features

Windows and Macintosh operating systems have many integrated accessibility features. End-users should explore these accessibility features before considering more expensive alternatives. The built-in accessibility features may be useful and sufficient for many end-users. Here is an abbreviated table of accessibility features built into Macintosh and Windows operating systems

Assistive Technology Matrix

Access Technology macOS Windows

High Contrast Accessibility → Display Ease of Access → Magnifier

Screen Reading Accessibility → VoiceOver Ease of Access → Narrator
